# 🚫 DisableSogouNetwork - 搜狗输入法网络连接管理工具

> 基于 [yongxin/DisableSogouNetwork](https://github.com/yongxin-ms/DisableSogouNetwork) 项目二次开发  
> 专注于安全、稳定地管理搜狗输入法的网络行为

---

## ✨ 核心功能亮点

### 🔍 智能检测系统
- **自动路径定位** - 智能识别搜狗输入法安装位置，无需手动进行选择
- **多版本支持** - 兼容搜狗输入法多个安装实例和不同版本
- **注册表防护** - 自动建立代理设置保护机制，防止配置被篡改

### 🛡️ 网络安全控制
- **代理穿透防护** - 有效阻止搜狗输入法绕过系统代理设置
- **网络连接隔离** - 彻底禁用不必要的网络请求，保护隐私安全
- **防火墙规则优化** - 确保网络策略正确应用且稳定持久

### 📈 用户体验优化
- **实时进度反馈** - 清晰展示每个步骤的执行状态和结果
- **详细操作日志** - 完整记录所有操作细节，便于问题排查
- **一键式操作** - 简化复杂配置流程，降低使用门槛
- **错误友好提示** - 针对常见问题提供明确的解决方案
- **不重复添加规则** - 对于已存在的规则不会再添加，只添加不存在的规则

## 🚀 快速开始

### 第一步：下载脚本
将 `run.ps1` 文件保存到本地目录

### 第二步：准备执行环境
1. **打开 PowerShell**：
   - 在脚本所在目录的地址栏中输入 `powershell` 后按回车
   - 或按住 Shift + 右键点击空白处，选择"在此处打开 PowerShell 窗口"

2. **检查并设置执行策略**（如遇权限错误）：
   ```powershell
   # 查看当前执行策略
   Get-ExecutionPolicy
   
   # 如果显示"Restricted"，需要临时更改策略
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
   
   # 确认更改（输入Y并按回车）
